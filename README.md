# 愛犬の鳴き声の感情翻訳ツール

愛犬の鳴き声をブラウザだけで録音・解析し、周波数特徴からおおよその感情（甘え・警戒・威嚇・遊びの誘いなど）を推定して表示するツールです。サーバーや外部APIには一切音声データを送信せず、すべてクライアントサイド（ブラウザ内）で完結します。

## 主な機能

- **録音**: `getUserMedia` でマイク入力を取得し、`AudioContext` に接続
- **周波数分解**: `AnalyserNode`（FFT）でリアルタイムに周波数成分へ分解
- **スペクトログラム描画**: 時間×周波数×強度を Canvas 2D API でヒートマップ的に描画し、鳴き声の周波数変化を可視化
- **特徴量抽出**: 基本周波数（ピッチ）、スペクトル重心、音量エンベロープ、継続時間、立ち上がりの鋭さなどをフレームごとに算出
- **感情分岐（ルールベース）**: 抽出した特徴量の閾値・パターンを JS の条件分岐で評価し、感情ラベル（例: 甘え / 警戒・威嚇 / 遊びたい / 不安）を推定して表示

## 技術スタック

- Vanilla JavaScript（ES Modules、フレームワーク・ビルドツール不使用）
- Web Audio API（`AudioContext` / `AnalyserNode` / `getUserMedia`）
- Canvas 2D API（スペクトログラム描画）
- HTML / CSS（UI）

外部ライブラリや npm 依存パッケージは現時点で使用していません。すべてブラウザ標準APIのみで実装します。

## セットアップ

マイク入力（`getUserMedia`）はブラウザのセキュリティ制約により `file://` で直接開くと動作しないため、簡易ローカルサーバーを起動して使用します。以下のどちらか一方で構いません。

```bash
# Node.js がある場合（インストール不要で実行可能）
npx serve .

# Python がある場合
python -m http.server 8000
```

起動後、表示された URL（例: `http://localhost:3000` や `http://localhost:8000`）にブラウザでアクセスしてください。

## 使い方

1. 上記のローカルサーバーを起動し、ブラウザでアクセスする
2. マイクへのアクセスを許可する
3. 「録音開始」ボタンを押し、愛犬に鳴いてもらう
4. リアルタイムに描画されるスペクトログラムを確認する
5. 鳴き終わると、抽出された特徴量に基づく感情の推定結果が表示される
