# 愛犬の鳴き声の感情翻訳ツール

愛犬の鳴き声をブラウザだけで録音・解析し、周波数特徴から感情（甘え・要求、警戒・威嚇、遊びの誘い、さみしさ・分離不安、痛み・SOS、挨拶・呼びかけ など）を推定して表示するツールです。サーバーや外部APIには一切音声データを送信せず、すべてクライアントサイド（ブラウザ内）で完結します。

## 主な機能

- **2つの入力方法**: その場でマイク録音するか、保存済みの音声ファイル（mp3 / wav など）を選ぶ
- **周波数分解**: `AnalyserNode`（FFT）でリアルタイムに周波数成分へ分解
- **スペクトログラム描画**: 時間×周波数×強度を Canvas 2D API でヒートマップ的に描画し、鳴き声の周波数変化を可視化
- **特徴量抽出**: 基本周波数（ピッチ）、スペクトル重心、音量エンベロープ、継続時間、立ち上がりの鋭さ、反復間隔、ノイズ成分などを算出（オフライン解析）
- **感情分類（ルールベース）**: 抽出した特徴量から感情ラベルを推定。犬の鳴き声と判断できない音（人の声・他の動物の声・環境音など）は無理に当てず「判定できません」を返す
- **スマホ対応**: iPhone / Android のブラウザで動作（マイク利用には HTTPS が必要。後述）

## 感情ラベル

甘え・要求 / 警戒・威嚇 / 遊びの誘い / 不安・恐怖 / さみしさ・分離不安 / 痛み・SOS / 挨拶・呼びかけ、および判定できない場合の棄却クラス。

> この判定は音の特徴に基づく参考程度の推定です。確立された科学的・獣医学的基準に基づくものではありません。また、犬以外の動物の声は正しく判定できません。

## 技術スタック

- Vanilla JavaScript（ES Modules、フレームワーク・ビルドツール不使用）
- Web Audio API（`AudioContext` / `AnalyserNode` / `AudioWorklet` / `getUserMedia`）
- Canvas 2D API（スペクトログラム描画）
- HTML / CSS（UI）

外部ライブラリや npm 依存パッケージは使用していません。すべてブラウザ標準APIのみで実装しています。

## セットアップ

マイク入力（`getUserMedia`）はブラウザのセキュリティ制約により `file://` で直接開くと動作しないため、ローカルサーバーを起動して使用します。

### PC で使う場合（HTTP で十分）

```bash
# Node.js がある場合（インストール不要で実行可能）
npx serve .

# Python がある場合
python -m http.server 8000
```

起動後、表示された URL（例: `http://localhost:3000`）にブラウザでアクセスしてください。

### スマホ実機で使う場合（HTTPS が必要）

スマホの `getUserMedia` は `http://<LAN-IP>` では動作せず、HTTPS が必須です。同梱の `https-server.js` が自己署名証明書でLAN内にHTTPS配信します。

```bash
# 1. 自己署名証明書を生成（OpenSSL が必要。<LAN-IP> は自分のPCのIPに置き換える）
mkdir -p .certs
openssl req -x509 -newkey rsa:2048 -nodes \
  -keyout .certs/key.pem -out .certs/cert.pem -days 365 \
  -subj "/CN=<LAN-IP>" \
  -addext "subjectAltName=IP:<LAN-IP>,DNS:localhost,IP:127.0.0.1"

# 2. HTTPS サーバーを起動（ポート8443）
node https-server.js
```

スマホを同じ Wi-Fi に繋ぎ、`https://<LAN-IP>:8443/` を開きます。自己署名証明書のため「安全でない接続」の警告が出ますが、「詳細 → このまま続行」で進めばマイクが使えます。Windows のファイアウォールで 8443 番の受信許可が必要な場合があります。

## 使い方

**マイクで録音する場合**

1. サーバーを起動し、ブラウザでアクセスする
2. 「録音開始」を押してマイクを許可し、愛犬に鳴いてもらう
3. リアルタイムに描画されるスペクトログラムを確認する
4. 「録音停止」を押すと、感情の推定結果が表示される
5. 「録音を再生」で録った音を確認できる

**音声ファイルを選ぶ場合**

1. 「ファイルを選択」から鳴き声の音声ファイル（mp3 / wav など）を選ぶ
2. 選ぶと自動で解析され、感情の推定結果が表示される
3. 「音声を再生」で音を確認できる
